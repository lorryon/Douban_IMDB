Introduction

This project aims to analyze movie ratings from Chinese and American viewers and draw insight on how the two markets differ in preferences.

Project Outline

  • Data Source
    Movie ratings for 7000 movies and TV series scraped from IMDB.com and Douban.com.
  • Data Processing
    Used Pandas DataFrame to clean, process and aggregate ratings data
  • Web App
    Built an interactive webapp using Flask for data visualization
  • Future Analysis
    Gather more features (director and cast popularity, box office rate) and apply predictive analysis models

Target Market

Help movie industry and entertainment providers make insightful decisions
